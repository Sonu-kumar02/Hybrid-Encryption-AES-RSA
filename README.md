# Hybrid Encryption Application (AES + RSA)

## 📌 Description
This project implements a GUI-based Hybrid Encryption system using AES and RSA algorithms. It ensures secure data transmission by combining the speed of AES with the security of RSA.

## 🔐 Features
- AES-based file encryption
- RSA-based secure key exchange
- GUI interface using Tkinter
- File encryption & decryption
- Secure hybrid encryption model

## ⚙️ Technologies Used
- Python
- Tkinter (GUI)
- PyCryptodome (Cryptography)

## ▶️ How to Run
1. Install dependencies:
   pip install pycryptodome

2. Run the application:
   python main.py

## 📂 Project Structure
- main.py → Main application
- public.pem → Public key
- private.pem → Private key (not uploaded for security)

## 💡 Concept
AES encrypts the data, and RSA encrypts the AES key. This ensures both speed and security.
